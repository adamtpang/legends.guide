# legends.guide - Visual & Audio Feature List

## Visual Features

### Digital Avatars
- [ ] AI-generated portrait for each founder (realistic, painterly style, consistent aesthetic)
- [ ] Animated idle state (subtle breathing/movement so they feel alive)
- [ ] Expression changes based on response tone (serious when giving tough advice, warm when encouraging)
- [ ] Full-screen avatar view during conversation (the founder "looks at you" while speaking)
- [ ] Gradient-to-portrait reveal animation on page load

### UI/UX Polish
- [ ] Framer Motion page transitions (smooth fade/slide between home, chat, compare)
- [ ] Framer Motion message animations (messages slide in, streaming text fades in)
- [ ] shadcn/ui component system (buttons, selects, cards, dialogs)
- [ ] Peterson Academy-inspired palette: warm ivory/cream backgrounds, deep navy/charcoal text, gold accents, aged paper textures
- [ ] Premium serif font for founder names/quotes (e.g. Playfair Display, Cormorant)
- [ ] Clean sans-serif for body text (Inter or similar)
- [ ] Typing indicator that matches each founder's personality (Munger: slow and deliberate, Musk: rapid-fire)
- [ ] Source citations styled as elegant footnotes with book cover thumbnails
- [ ] Mobile-first responsive design
- [ ] Dark/light mode toggle (dark default)

### Compare Page
- [ ] Side-by-side avatar portraits during comparison
- [ ] Visual "debate" feel with founder avatars facing each other
- [ ] Highlight differences in philosophy with color-coded annotations

## Audio Features

### Voice Clones (ElevenLabs)
- [ ] Custom voice clone per founder matching their real speaking style
- [ ] Rockefeller: slow, deliberate, paternalistic, slight period affect
- [ ] Jobs: intense, simple words, dramatic pauses, California accent
- [ ] Bezos: precise, punctuated by his distinctive laugh
- [ ] Musk: halting, think-out-loud cadence, South African traces
- [ ] Jensen: passionate, emotional, storytelling energy
- [ ] Thiel: measured, philosophical, German-American precision
- [ ] Munger: dry wit, pithy, Midwestern bluntness
- [ ] Franklin: warm, conversational, colonial New England charm
- [ ] Walton: folksy, enthusiastic, Arkansas drawl
- [ ] Naval: calm, aphoristic, measured pauses between insights

### Audio UX
- [ ] Play/pause button on each message (speaker icon)
- [ ] Auto-play toggle (hear responses as they stream in)
- [ ] Audio waveform visualization during playback
- [ ] Volume control
- [ ] Speed control (0.75x, 1x, 1.25x, 1.5x)

## Future / WhatsApp Integration
- [ ] Each founder gets a WhatsApp Business number
- [ ] Users can text any founder and get in-character responses
- [ ] Voice messages supported (STT -> response -> TTS)
- [ ] Marketing: "Text Rockefeller at +1-xxx-xxx-xxxx for business advice"
- [ ] Conversation continuity across web and WhatsApp
